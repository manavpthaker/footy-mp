**SectionHeading** — uppercase divider with an accent tick that opens each analytical section in a detail/match view.

```jsx
<SectionHeading>Full squad — ★ = key player</SectionHeading>
<SectionHeading tick="var(--accent-2)" trailing="69/99">Model rating</SectionHeading>
```

Props: `tick` (color of the leading bar), `trailing` (right-aligned value/count). Spacing is built in (22px top / 11px bottom).
