**Tag** — status chip carrying tournament meaning. Use on standings rows, hero headers, and bracket slots.

```jsx
<Tag tone="qualified">Q</Tag>
<Tag tone="out">OUT</Tag>
<Tag tone="pill">1st in Group A</Tag>
<Tag tone="accent">Round of 16</Tag>
<Tag tone="gold">★ Key</Tag>
```

Tones: `qualified` (green), `out` (red), `pill` (neutral), `accent` (sky-blue rounded), `gold` (champion). Keep labels terse and uppercase for status (Q / OUT).
